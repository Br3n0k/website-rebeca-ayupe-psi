INSTRUÇÕES PARA ADICIONAR A FOTO DA DRA. REBECA AYUPE

1. Coloque a foto da Dra. Rebeca Ayupe na pasta "images/" com o nome "rebeca_ayupe.jpg"
2. Execute o comando "bun run build" na raiz do projeto
3. Faça o upload da pasta "public_html/" para o servidor Apache da Hostinger

Dessa forma, a foto aparecerá na seção principal e na seção "Sobre Mim" do site.
